Browser-side React app. A diary of films watched, including watch date. Sorting and flitering the diary supported.

Limited functionality and future fixes:
Currently can only add films from a local database of 30 films. Requests to The Movie Database will be implemented.
The Favorites bar not implemented. Drag-and-drop feature from diary to favorites will be implemented.

App can be iniated with NodeJS with the following commands:
npm init

npm run build

serve -s build
